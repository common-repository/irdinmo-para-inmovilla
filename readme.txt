Irdinmo para Inmovilla es un plugin sencillo de usar, pero potente. En menos de dos minutos tendrán conectada tu web Wordpress a tus propiedades de Inmovilla.

== Description ==

En menos de dos minutos tendrán conectada tu web Wordpress a tus propiedades de Inmovilla.

Simplemente rellena tu código de agencia y tu contraseña en las opción de menú API Inmovilla. Si no sabes esta información, debes solicitarla a soporte de Inmovilla

Revisa el resto de parámetros de configuración y cambia lo que consideres

Pega el shortcode en cualquier punto de tu web

Listo!

Notas: 
Este plugin hace uso de la API pública de Inmovilla: https://apiweb.inmovilla.com/apiweb/apiweb.php
Inmovilla es un software externo y con el que no tenemos ninguna relación comercial. 
El acceso a la información contenida en Inmovilla es responsabilidad exclusiva de dicha compañía, por lo que para cualquier información que compartas con esta compañía, previamente has de firmar un contrato con ellos y ser conocedor de su política de privacidad: https://inmovilla.com/politica-de-privacidad/
Para más información acerca de tarifas y contratación de inmovilla, dirígete a su página web: https://inmovilla.com o su correo soporte@inmovilla.com.
Para el uso del plugin Irdinmo para Inmovilla (edición gratuita) no es necesario compartir ninguna información con https://inforrada.es
Este plugin hace uso de la API pública de Leaflet: https://api.mapbox.com
Leaflet es un software externo y con el que no tenemos ninguna relación comercial.

== Frequently Asked Questions ==
= ¿Tiene el plugin en su versión gratuita alguna publicidad? =
No. Solamente tiene menos funcionalidades que la versión premium.

= ¿Será siempre gratuito? =
Sí. 

= ¿Cada cuanto recibiré actualizaciones de la versión gratuita? =
La versión gratuita irá siempre un año por detrás de las actualizaciones de la versión premium. Como excepción, siempre que encontremos un bug, lo resolveremos en ambas versiones.


== Installation ==
Descarga Irdinmo para Inmovilla en tu página web, actívalo e introduce tu usuario y contraseña de Inmovilla. Selecciona el idioma.

1, 2, 3: Hecho!
